## Terms To Avoid Or Deprioritise

| Term | Reason |
|---|---|
| `midi` | Ambiguous and informational; not a collection-page target. |
| `midi dress sale`, `sale midi dress`, `midi dresses for sale` | Sale intent should sit on sale-specific URLs unless this page is actively merchandised as sale. |
| `midi skirt`, `midi skirts`, `midi skirt australia` | Sibling skirt-page intent. Do not use for the dress page except as cannibalisation examples. |
| `printed midi dress` | Better mapped to Printed Dresses unless this page is filtered to printed midis. |
| `floral midi dress`, `womens floral midi dress` | Fresh demand exists, but live product sample did not show floral-led midi products. |
| `linen midi dress`, `knitted dress midi`, `sweater midi dress`, `sequin midi dress` | Fabric/style mismatch in the current sample. |
| `pleated midi dress`, `smocked midi dress`, `midi slip dress` | Evaluator accepted them, but current product evidence is not strong enough to recommend. |
| `plus size midi dress`, `plus size midi dresses` | Catalogue mismatch; Melani does not appear to support this intent. |
| Retailer/competitor terms such as `zara midi dress`, `myer midi dress`, `david jones midi dress`, `kmart midi dress` | Brand/retailer contamination. |

## SEO Implementation Notes

| Element | Recommendation |
|---|---|
| Title tag | `Midi Dresses | Formal & Occasion Midis | Melani the Label` |
| H1 | `Midi Dresses` |
| Meta description | Lead with Australian-made midi dresses, fitted/stretch construction, and formal/occasion styling. Include free shipping only if the current offer remains active. |
| Body copy angle | Fit and fabrication first: stretch, lined, gathered, lace, multi-neckline, occasion-ready, formal, and colour range. Keep the tone precise and product-led. |
| Internal links | Link to `all-dresses`, `gowns`, `long-sleeve`, `lace-dresses`, and `bridal` where copy mentions broad dresses, formalwear, sleeves, lace, or white/bridal intent. |
| Cannibalisation guard | Do not over-optimise this page for long-sleeve, lace, bridal, printed, floral, sale, or skirt queries. Use those as internal-link paths to sibling collections. |

## Final Recommendation

Proceed with the Midis collection as a high-priority keyword target:

- Primary: `midi dress`
- H1/editorial wording: `Midi Dresses`
- Core secondaries: `midi dresses`, `midi dress for women`, `midi dress australia`, `midi party dresses`, `formal midi dress`
- Conditional secondaries: `midi dress with sleeves`, `black midi dress`, `white midi dress`, `long sleeve midi dress`, `lace midi dress`, `red midi dress`

No SE Ranking keyword writes were made. The main Midi terms are already tracked in project `12019802`; the next useful step is to use this repaired set when rebuilding the Melani collection content brief for `midis`.
