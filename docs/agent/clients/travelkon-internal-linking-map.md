## Reusable Linking Rules

1. Link from relevant blogs to the strongest matching category hub first.
2. Add one product link only where the article naturally supports a purchase decision.
3. Use Search Console query language for anchors, but avoid exact-match repetition. Rotate between primary query, product/category wording, and natural variants.
4. Keep each article focused: 2-4 internal links for short articles, 4-7 for long guides.
5. Do not link from cart, checkout, account, upload, pagination, legal, or admin URLs.
6. For destination content, link upward to the destination category, sideways to a SIM/eSIM comparison guide where relevant, and downward to priority products.
7. For support content, link to setup/troubleshooting pages plus the destination product/category only when the support problem is country-specific.
