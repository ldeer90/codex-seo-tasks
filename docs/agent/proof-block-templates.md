# LD SEO Proof Block Templates

Proof blocks belong in agent responses, local validation artifacts, or approved internal Monday updates when appropriate. They must not be pasted into client-facing Google Docs, Sheets, or polished client comments.

Use these templates after the routed workflow has passed its validation/readback gates. Keep values concise and evidence-backed.
